* Steps:
1. Install modules
 ```
 npm install
 
 // if you're using Yarn
 yarn install
 ```
2. Edit table info and API info in ```tableInfo.js``` and ```Texts.js``` in ```data``` folder. To check live preview, run:
```
npm dev // check port 8080
```
3. When finished, run:
```
npm build
```
4. Done!
